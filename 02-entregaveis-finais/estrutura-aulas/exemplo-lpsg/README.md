---

## 🎯 Como usar este exemplo

1. Lê o `00-variaveis-preenchidas.md` pra entender o **modelo de preenchimento**
2. Compara com seu próprio nicho/produto pra validar se a estrutura encaixa
3. Volta no `template/00-variaveis-globais.md` e preenche com **seus** dados
4. Itera nos arquivos das 6 aulas
