Como rodar em servidor local

        
Necessario para o Google Sheets ao vivo funcionar

      

      
    


    

      Quando voce abre o arquivo com duplo-clique, o navegador usa o protocolo file:// — e o Google bloqueia conexoes vindas desse protocolo. Rodar em servidor local (http://localhost) resolve.
    


    

      

        
Passo 1 — Abra o Terminal

        
Mac: Cmd + Espaco → digite "Terminal" → Enter
Windows: Win + R → "cmd" → Enter

      


      

        
Passo 2 — Va ate a pasta do arquivo

        

          cd "PASTA_DO_ARQUIVO"
          
        

        
No Mac voce pode arrastar a pasta para o terminal — substitui o comando cd.

      


      

        
Passo 3 — Inicie o servidor

        

          python3 -m http.server 8765
          
        

        
Python vem instalado no Mac. Windows: baixe em python.org se nao tiver.

      


      

        
Passo 4 — Abra no navegador

        

          http://localhost:8765/dashboard-lancamento-lpsg-template.html
          
        

      


      

        Pronto. Com o dashboard aberto em http://localhost, o Google Sheets ao vivo funciona normalmente — incluindo URLs de compartilhamento /edit?usp=sharing.

Template Turbo Academy — LPSG

  

  


    
    

      

        
Nível 1 — Uso básico (zero código)

        
Planilha oficial + importação direta

        
Cada linha da planilha oficial de dados é um lançamento completo. Preencha, importe e o dashboard mostra todos os lançamentos com seletor e modo comparação.


        

          
1

          

            
Clique em "Baixar modelo" no topo

            
Baixa o arquivo planilha-lpsg-modelo.xlsx com a estrutura oficial: linha 1 com títulos de bloco, linha 2 com cabeçalhos, linha 3 com exemplo de dados, demais linhas vazias para preencher.

          

        

        

          
2

          

            
Preencha uma linha por lançamento

            
Cada linha da aba Dados dos Eventos é um lançamento. Coluna A = identificador (ex: LPSG_01/26). Os blocos são: Captação / Tráfego / Pico Ao Vivo / Espectadores Únicos / Conversão / Resultado. Use os formatos que já estão na planilha — R$ e % são aceitos.

          

        

        

          
3

          

            
Importe a planilha

            
Clique em "Importar planilha" no topo ou arraste o arquivo .xlsx para a janela. O dashboard lê todos os lançamentos, ignora linhas sem identificador e converte R$/% automaticamente.

          

        

        

          
4

          

            
Selecione o lançamento e compare

            
Na aba Métricas, um seletor no topo mostra todos os lançamentos importados. Clique em "Comparar" para ver 2 ou 3 lançamentos lado a lado com setas de evolução.

          

        


        


        
Estrutura da planilha oficial

        

          
Dados dos Eventos

          

            

              
Linha 1CAPTAÇÃOTRÁFEGO...

              
Linha 2$ IngressoIngressosCACCPMCTR...

              
LPSG_01/26R$ 62,001.000R$ 49,60R$ 31,003,00%...

              
LPSG_02/26← seu próximo lançamento aqui

            

          

          

            Linha 1 = títulos de bloco (mesclados). Linha 2 = cabeçalhos reais. Linha 3+ = dados (um lançamento por linha). O dashboard filtra linhas sem identificador na coluna A.
          

        

      

    


    
    

      

        
Nível 2 — Google Sheets ao vivo

        
Dashboard conectado na sua planilha

        
Mantém os dados no Google Sheets e o dashboard puxa direto. Atualiza a planilha, atualiza o painel.


        

          Pre-requisito: rode o dashboard em servidor local — nao funciona com duplo-clique. 
        


        

          
1

          

            
Compartilhe a planilha

            
Na sua planilha, va em Arquivo → Compartilhar → Acesso geral. Escolha Qualquer pessoa com o link → Leitor. Copie o link gerado — o dashboard aceita qualquer formato de URL do Google Sheets.

          

        

        

          
2

          

            
Cole no dashboard e conecte

            
Clique em Google Sheets no topo do dashboard. Cole a URL no campo. Use Testar conexao para confirmar, depois clique em Conectar e sincronizar. A partir de agora, voce atualiza so a planilha — o dashboard reflete sozinho.

          

        

        

          
3

          

            
Ative o auto-sync (opcional)

            
Marque Atualizar automaticamente a cada 5 minutos antes de conectar para manter o painel atualizado enquanto a aba estiver aberta.

          

        

        

          
Alt

          

            
Metodo alternativo (se o passo 2 falhar)

            
Publique via Arquivo → Compartilhar → Publicar na web → CSV e cole a URL gerada. Ela comeca com /d/e/2PACX-....

          

        

      

    


    
    

      

        
Nível 3 — Claude Code

        
Atualização via Claude

        
Se quiser que o Claude leia a planilha e atualize o JSON do dashboard, clique em "Claude Code" no topo para copiar o prompt pronto.

        

          
1

          

            
Clique em "Claude Code" no topo

            
Copia o prompt automático. Abra o Claude Code (CLI ou claude.ai), cole o prompt e anexe a planilha XLSX. O Claude identifica as colunas pelo padrão da planilha oficial e retorna o JSON atualizado para substituir no bloco dashboard-data deste arquivo.
