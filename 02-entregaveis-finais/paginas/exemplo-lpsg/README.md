# Exemplo preenchido — Páginas LPSG do Leo Tabari

> Versão exemplo do template de páginas com dados reais do LPSG do Leo Tabari (Turbo Academy).

---

## 📁 Arquivos

```
exemplo-lpsg/
├── README.md                       ← você está aqui
└── 00-variaveis-preenchidas.md     ← todas as variáveis com dados reais + 5 variações preenchidas
```

> Os arquivos 01-05 do template (`01-stack-e-deploy.md`, `02-arquitetura-projeto.md`, etc.) **não foram duplicados** — a estrutura técnica é idêntica entre projetos. Use o template + as variáveis preenchidas aqui.

## 🔑 Diferença template genérico → exemplo LPSG

| Variável | Template | Exemplo LPSG |
|---|---|---|

## 🚀 Como usar

1. **Abra o template em `../template/`**
2. **Use `00-variaveis-preenchidas.md` deste exemplo como referência** ao preencher suas próprias variáveis
3. **Adapte ao seu nicho** — a estrutura é a mesma, os dados mudam
4. **Compare suas métricas** com os benchmarks do LPSG real
